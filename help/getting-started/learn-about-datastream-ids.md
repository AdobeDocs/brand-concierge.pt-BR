# Saiba mais sobre IDs de fluxo de dados

Saiba como a ID de sequência de dados conecta os eventos do Web SDK do seu site ao Adobe Brand Concierge no Adobe Experience Platform. Você sabe por que a ID é necessária, como ela roteia dados para a sandbox e os conjuntos de dados corretos e como as equipes usam fluxos de dados separados para desenvolvimento, preparo e produção.

## Para quem é este vídeo?

* Desenvolvedor
* Gerente de produto
* Consultor de soluções

## Conteúdo de vídeo

* Por que a ID do fluxo de dados é necessária para o Web SDK e o Brand Concierge
* Como o fluxo de dados roteia eventos para sandboxes, conjuntos de dados, esquema e identidades
* Por que as equipes usam fluxos de dados diferentes para desenvolvimento, preparo e produção
